#Dynamic Cards

### Description:

This site builds flashcards based on user input entered into a text box. A card appears when the user clicks the submit button.  Each card has a self-contained delete button that, when clicked, removes the card from the site.

### Technologies used:

- HTML
- CSS
- Javascript

## To Run:
1. git clone https://github.com/aefinch/DynamicCards
1. cd DynamicCards
1. Run http-server
1. Navigate in browser to http://localhost:8080
